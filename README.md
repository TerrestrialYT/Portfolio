A premium, high-converting, single-page personal portfolio website designed for a financial professional and Mutual Fund/SIP Distributor. Built with a modern "fintech" aesthetic featuring deep dark modes, bullish emerald gradients, and a typography-driven layout.

## ✨ Features
* **Premium Fintech UI/UX:** Deep midnight backgrounds with CSS-generated trading grid overlays and glowing emerald gradients.
* **Typography-Driven Design:** Zero reliance on external images or placeholders. Fast, clean, and entirely self-contained.
* **Fully Responsive:** Mobile-first layout optimized for all screen sizes using Tailwind CSS.
* **Smooth Animations:** Buttery-smooth scroll reveal effects powered by AOS (Animate On Scroll).

* **HTML5:** Semantic markup for structure.

* **AOS Library:** Scroll animation logic (via CDN).
* **Google Fonts:** 'Outfit' (Headings) and 'Inter' (Body).

## 🚀 Deployment (GitHub Pages)
This project is built as a pure, single-file static website (`index.html`) with no build steps or bundlers required, making it incredibly easy to host on GitHub Pages.

1. Create a new repository on GitHub and upload the `index.html` file.
2. Go to your repository **Settings**.
3. Navigate to **Pages** in the left sidebar.
4. Under "Build and deployment", set the **Source** to `Deploy from a branch`.
5. Under "Branch", select `main` (or `master`), leave the folder as `/ (root)`, and click **Save**.
6. Wait 1-2 minutes, and your live URL will appear at the top of the Pages settings screen.
